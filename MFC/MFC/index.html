消息映射

消息映射机制

  • 声明宏 写在.h中类中 DECLARE_MESSAGE_MAP
  • 分界宏 写在.cpp中
    • BEGIN_MESSAGE_MAP( theClass, baseClass )
    • END_MESSAGE_MAP( )
  • 消息宏：写在 分界宏 中间
    • 鼠标左键按下 ON_WM_LBUTTONDOWN()
    • 键盘 ON_WM_CHAR()
    • 绘图 ON_WM_PAINT()
  • 函数原型 声明写到 .h中 （afx_msg 没有作用，是一个标识）
    • 鼠标左键按下 afx_msg void OnLButtonDown(UINT, CPoint);
    • 键盘 afx_msg void OnChar(UINT, UINT, UINT);
    • 绘图 afx_msg void OnPaint();
  • 函数实现写到 .cpp 中

Windows字符集

1）多字节（8位的ANSI字符集）：1个字符对应1个字节

2）宽字节（16位Unicode字符集）：ANSI字符集的超集，支持中文，日文等

3）TEXT (_T)宏

MFC中的TEXT宏可以自动适应字符类型，如果定义了预处理器程序符号_UNICODE，那么编译器将使用Unicode字符，如果没用定义该预处理器程序符号，那么编译器将使用ANSI字符。

4）TCHAR类型

如果定义了_UNICODE符号TCHAR将变为wchar_t类型。如果没用定义_UNICODE符号，TCHAR将变为普通古老的char类型。
