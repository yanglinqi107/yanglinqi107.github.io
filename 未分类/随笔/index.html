VM虚拟机CentOS固定IP-桥接模式

VMware虚拟机通过bridged(桥接模式)设置静态IP_Helele_Blog的博客-CSDN博客_虚拟机桥接模式静态ip

1）查看本机IP信息

控制面板 -》网络和Internet -》网络和共享中心

2）修改CentOS的网卡配置文件

3）测试是否能上网，可以ping百度或网关

注意：

设置静态IP发现无法上网，可能是该ip已经被使用了。

  1. 可以先使用dhcp获取一个随机IP，用hostname -I查看IP，再设为static的IP
  2. 在子网的范围下手动修改ip，只能碰运气了

局限：桥接模式是从路由器获取IP上网，所以换一个网络可能又需要修改网关、ip掩码等，所以如果是用的是宽带或同一个WIFI可以使用桥接模式固定IP。

pip升级：python3 -m pip install —upgrade pip
